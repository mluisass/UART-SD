# UART - SD
## Como usar esse repositório?
### Clonar
```sh
$ git clone 'https://github.com/mluisass/UART-SD'
```
### Pegar últimas alterações (pull)
```sh
$ git pull --rebase
```
### Inserir suas alterações no repositório
1. Adicionar 
```sh
$ git add .
```
2. Fazer commit
```sh
$ git commit -m 'mensagenzinha'
```
3. Empurrar para o repositório (push)
```sh
$ git push --force
```
